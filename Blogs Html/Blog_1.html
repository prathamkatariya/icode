Make Your Chrome Extension in Just 3 Simple Steps

Making a chrome extension is as simple as making a website in HTML, CSS and Javascript.
without wasting anytime lets start making our extension.

Preview of Our Extension.

-> Step-1 : Creating a manifest file.

Create a manifest.json file and write the code given below.

{
    "manifest_version":2,
    "name":"Keep Notes",
    "version":"1.0",
    "description":"Create notes easily",

// size of icons that are displayed on chome browser.

"icons":{
    "128":"images/icon128.png",
    "48":"images/icon48.png",
    "16":"images/icon16.png"
},

// on clicking the icon following default action takes place

"browser_action":{
    "default_popup":"index.html",
    "default_icon":"images/icon48.png",
    "default_title":"Keep Notes"
},

// We need this extension on all the tabs so here we ask for permissions

"permissions":[
    "tabs"
]
}

-> Step-3 : Publishing it locally

1. Click on 3 dots.



2. Click on More tools and then Extensions.
3. Enable developer mode.



4. Click Load unpacked and select the folder, then upload that folder.



5. Now You are Ready to use your Extension.
